Finals Lab Task 6

This task focuses on performing fundamental MongoDB operations such as creating a database, inserting documents, and executing find, update, search, and delete actions within a collection.

Step-by-Step Process:

  1. Create a Database:
    • Generate or select a desired MongoDB database to serve as the environment.
  2. Insert Documents:
    • Create a collection within the database.
    • Add multiple documents containing fields like name, age, or address.
  3. Find Documents:
    • Retrieve all documents from the collection.
    • Apply filters to find specific documents by name or ID.
  4. Update Documents:
    • Locate documents using conditions (e.g., name).
    • Modify field values and confirm the update.
  5. Search Documents:
    • Query documents using criteria like age or specific text values.
  6. Delete Documents:
    • Identify documents to remove and delete them from the collection.
    • Verify deletion by listing remaining documents.
